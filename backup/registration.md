# 简介
此项目是由LJT Studio开发的一个专用于运动会检录的项目，历时10天建成beta->1.0.0版本（5.1-5.10），此项目采用远程数据库来确保数据安全，并在插入和删除数据的后端添加用户名和密码验证，以保证用户数据安全
# 主要功能
1.通过前端的连接数据库来实现查找运动员的名字
2.后端获取到用户名和密码数据库，实现登录功能
3.当用户输入正确启动程序否则提示错误
4.我们使用基于.NET framework架构的c#语言和MySql.Data NuGet程序包
5.我们使用advanced installer来打包安装程序
6.使用此程序请安装.NET framework4.0及以上的依赖
7.我们支持Windows XPsp2到Windows1124h2版本（包括x32和x64的处理器）
