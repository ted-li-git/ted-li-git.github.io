# 前言
相信大家改系统文件时都遇到过权限不足的情况，但我们却很难拿到权限（一般Windows文件夹里的文件或文件夹都需要administrator或system甚至TrustedInstaller）所以提权是一个很重要的工具
# 工具
我今天给大家推荐的是Nsudo，他界面简洁，用处大，bug少。
Nsudo下载链接：
1.github开源（建议大家从这里下载，支持原作者）：https://github.com/M2TeamArchived/NSudo
2.蓝奏云：https://lijunyu.lanzn.com/ilMDl2uh1naj
> [!IMPORTANT]
> 从github下载需要注意要下载9.0Preview1，8.2有bug。
## 用法
解压后找到你的文件夹，打开x64文件夹（如果是32位系统就选Win32，ARM64就选ARM64）在里面找到NsudoLC.exe运行即可。
