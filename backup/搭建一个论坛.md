# 买云服务器
买完域名就该买云服务器了，这里我用的雨云的云服务器，地区香港，系统是Ubuntu，预装了宝塔面板9.0。
# 论坛系统
我编程经验其实不多，就只会点c++和c#，HTML，CSS，JS，PHP都不大懂，于是我用了很有名的Discuz!来搭建论坛。
# 步骤
1.进入宝塔面板
2.在应用商店里选一键部署，搜索discuz!
3.一键部署discuz!（只用管域名，其他不动，注意看PHP），输入你的域名（这里记得在cloudflare上创建A记录，指向你服务器的IP）
4.部署完成后输入http://你的域名.你的后缀
5.~按常理来说~出现协议界面，点击我同意就行
6.后面让你输入数据库信息，地址不变，数据库名填写在宝塔面板里数据库的数据库名，用户名也在和数据库名同样的位置，密码也在那里
7.点击安装等待安装完成就可以开始食用了
