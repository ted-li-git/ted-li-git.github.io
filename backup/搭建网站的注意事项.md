# 注意事项
## 用github pages功能生成的网站
注意：
1.每天备份储存库
这个防止某天你忘记了账号，或github打不开了
解决方法：在桌面上创建一个批处理文件，内容如下：
```bash
cd /d C:\repo-name
git pull
```
> TIP:要提前安装好git:git-Windows下载链接:https://git-scm.com/downloads/win

# 用服务器（有宝塔面板）搭建的网站
注意：
1.设置计划任务
打开你服务器的宝塔面板，找到计划任务，点击添加任务，在任务类型里选备份网站/备份数据库，有条件的可以选数据库增量备份。
>TIP:记得下载备份到本地，以免需要清空数据
