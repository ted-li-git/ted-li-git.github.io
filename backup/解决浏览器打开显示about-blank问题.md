# 前言
大家可能遇到过这种情况，这个问题一般是因为浏览器的起始页面导致的，解决方法如下
# 解决
## Google Chrome
打开Chrome（此操作无需使用Google账户）点开右上角三个点，找到设置，点进去，点击起始页面，选择打开新标签页，重启Chrome，问题解决。
## Microsoft edge
打开edge，点开右上角三个点，点击设置，点击启动、点击主页和新建选项卡页，在启动时栏目下，选择打开新标签页，重启edge，问题解决。
