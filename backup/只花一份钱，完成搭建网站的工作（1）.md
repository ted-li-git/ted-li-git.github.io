# 前言
众所周知，搭建网站要域名和服务器，要花两份钱，虽然现在有不少免费提供的服务器，但配置很差或试用期短，但是你有没有想过，我们可以不花钱买或租服务器，只需要域名的钱，没错，GitHub pages要登场了！
# 准备工作
电脑（能上网）、GitHub（[请认准官方网站](https://github.com)）、GitHub的加速器（可以在[github快速入门（注册登录篇）](https://blog.ted-lijunyu.top/post/github-kuai-su-ru-men-%EF%BC%88-zhu-ce-deng-lu-pian-%EF%BC%89.html)里找到并下载安装）。
# 第一步：创建储存库与开启GitHub pages功能
OK，现在到了第一步，创建储存库，这里就不详细讲了，如需具体内容请参考[github快速入门（创建储存库篇）](https://blog.ted-lijunyu.top/post/github-kuai-su-ru-men-%EF%BC%88-chuang-jian-chu-cun-ku-pian-%EF%BC%89.html)
## 创建储存库与开启GitHub pages功能
1.点击New。
2.Repository name填`xxx.github.io`，xxx是你的用户名，Description不用填，一定勾选Add a README file。
3.点击create repository，创建储存库。
4.在Top repositories里找到你刚创建的储存库，单击进入。
5.在上方找到settings，点击pages，把Branch下的None改成main，然后点击Save。
6.完成！（这时你在浏览器新建一个窗口输入你的储存库名字就可以看见你的README.md里的内容啦）
# 第二步：上传HTML代码
既然可以看到README.md里的内容了，那一个README.md怎能满足我们，接下来让我们来上传HTML代码来解锁更多功能吧！更多操作见[github快速入门（新建/上传文件篇）](https://blog.ted-lijunyu.top/post/github-kuai-su-ru-men-%EF%BC%88-xin-jian---shang-chuan-wen-jian-pian-%EF%BC%89.html)
## 上传HTML代码
1.返回Code界面，依次点击Add file->Upload files->choose your files，找到你的HTML代码，等待进度条跑完，最后点击Commit changes，就大功告成了
> [!TIP]
> HTML的文件名一定要是index.html不然不会显示在主界面
